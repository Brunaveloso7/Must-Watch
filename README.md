# Must-Watch
Criando um sistema em Python utilizando o framework Flask para cadastrar, consultar, alterar e apagar registros de uma lista pessoal de filmes e séries do tipo "must-watch".

#  Must-Watch

Projeto desenvolvido com Flask para gerenciamento de filmes e séries.

##  Funcionalidades

- Adicionar filmes
- Editar filmes
- Excluir filmes
- Interface estilizada com CSS

##  Tecnologias utilizadas

- Python
- Flask
- HTML
- CSS

##  Como executar o projeto

1. Instale as dependências:
   pip install -r requirements.txt

2. Execute:
   python app.py
